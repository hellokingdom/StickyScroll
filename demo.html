StickyScroll Demo

So your sidebar can follow anywhere you go

Explanation: In this demo, the element we want to fix is called .fixme, and we want the element to stay within it's sidebar #sidebar-1, so we the code used is $('.fixme').stickyScroll({ container: '#sidebar-1' }).

Important: the fixme element and containing div must be clearfixed.
